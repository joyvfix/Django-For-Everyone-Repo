**Jawaban Soal 1:** Perintah apa untuk membuat proyek Django baru bernama 'mysite'?

> ✅ django-admin startproject mysite

---

**Jawaban Soal 2:** Perintah apa untuk membuat app baru bernama 'blog' di dalam proyek?

> ✅ python manage.py startapp blog

---

**Jawaban Soal 3:** Di port berapa Django development server berjalan secara default?

> ✅ Port 8000 — akses via http://127.0.0.1:8000/

---

**Jawaban Soal 4:** Apa fungsi file manage.py?

> ✅ Command-line utility untuk mengelola proyek Django: runserver, migrate, createsuperuser, shell, dll.
