## 🌅 Checklist Harian

- [ ] Desain schema database di kertas sebelum mulai coding model
- [ ] Selalu tambahkan __str__ method di setiap model
- [ ] Berikan verbose_name yang jelas untuk setiap field
- [ ] Jalankan python manage.py shell untuk test query sebelum tulis di view
- [ ] Backup database sebelum migrasi besar

## 🧠 Growth Mindset

> _"Saya belum bisa melakukan ini **BELUM**."_

Kalau kamu stuck, itu normal. Semua developer pernah stuck.
Yang membedakan adalah: **apakah kamu berhenti atau terus mencoba?**

**Ketika stuck:**
1. ⏸️ Istirahat 10 menit — jalan-jalan, minum air
2. 🔍 Baca ulang error message dengan pelan
3. 🌐 Google dengan kata kunci spesifik
4. 📝 Tulis ulang masalahnya dengan kata-kata sederhana
5. 💬 Tanya di komunitas — tidak ada pertanyaan yang bodoh
