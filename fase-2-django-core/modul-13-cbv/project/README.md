## 🚀 Cara Memulai

```bash
# Masuk ke folder project
cd project/src/

# Buat virtual environment (kalau belum)
python -m venv venv
source venv/bin/activate    # Mac/Linux
venv\Scripts\activate       # Windows

# Install dependencies
pip install -r requirements.txt

# Jalankan project
python manage.py runserver   # untuk Django project
# atau
python main.py               # untuk pure Python
```
