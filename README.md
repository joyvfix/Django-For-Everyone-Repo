## 🚀 Cara Mulai Belajar

### Langkah 1 — Clone Repo
```bash
git clone https://github.com/ahmadfaqih/django-for-everyone.git
cd django-for-everyone
```

### Langkah 2 — Baca Panduan Persiapan
```bash
# Baca README di Modul 01
cat fase-1-fondasi-python/modul-01-perkenalan/README.md
```

### Langkah 3 — Install Dependencies
```bash
python -m venv venv
source venv/bin/activate        # Mac/Linux
venv\Scripts\activate           # Windows
pip install -r requirements.txt
```

### Langkah 4 — Mulai dari Modul 01!
Buka folder `fase-1-fondasi-python/modul-01-perkenalan/` dan ikuti instruksinya.
